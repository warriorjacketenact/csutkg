# VideoNav 技术资源导航站

VideoNav 是一个面向开发者和技术研究人员的结构化外链资源聚合平台，专注于收集、分类与索引互联网上的高质量技术视频、教程、讲座及会议录播内容。项目定位于解决技术学习过程中资源分散、检索效率低、质量参差不齐的痛点，通过人工筛选与标签化组织，为使用者提供清晰、可追溯的技术视频资源入口。

项目采用静态站点生成架构，所有资源数据以结构化文本存储，支持快速检索与按需导出。VideoNav 适用于个人学习路径规划、团队技术分享库搭建以及教育机构课程素材采集等场景。当前批次为第 726/1241 批资源入库，共计收录 250 个经过核验的视频资源链接。

## 功能概览

**多维度标签过滤**：每个资源均附带技术领域、难度等级、时长区间、发布方类型等标签，支持组合筛选，帮助用户快速定位符合自身需求的学习材料。

**资源状态标注**：对每个外链进行可用性检测，标注资源状态，包括有效、需代理访问、源站变更等，减少无效点击。

**本地化缓存预览**：对视频标题、简介、发布者等元信息进行本地化存储，支持离线检索与元数据浏览，降低对外部服务的依赖。

**自定义收藏夹**：用户可创建个人收藏列表，将感兴趣的资源归类存档，支持导出为 JSON 或 Markdown 格式，便于分享与备份。

**定时更新机制**：内置调度任务，定期对已收录链接进行可用性重检，自动标记失效链接并生成报告，保证资源库的长期有效性。

**开放数据导出**：支持将整个资源索引导出为 CSV 或 SQLite 数据库文件，方便二次开发或与其他工具集成。

## 应用场景

技术团队内部培训选材：团队技术负责人可使用 VideoNav 筛选特定主题的视频资源，快速组建培训材料清单，分配成员学习并跟踪进度，降低课程设计的时间成本。

个人开发者系统学习：开发者可根据自身技术栈，利用标签过滤系统查找从入门到进阶的系列视频，形成系统化的学习路径，避免碎片化信息干扰。

高校课程辅助教学：计算机相关专业教师可将 VideoNav 作为课程资源补充工具，选取高质量技术讲座视频作为课外拓展材料，丰富教学内容维度。

技术会议资料归档：会议组织者或参会者可使用本平台对大会各分会场的视频链接进行统一归集，便于会后的资料整理与回看。

## 快速开始

以下命令可在本地环境完成 VideoNav 项目的克隆、依赖安装与开发服务器启动。

```bash
git clone https://github.com/videonav/videonav.git
cd videonav
npm install
npm run dev
```

执行完毕后，访问控制台输出的本地地址即可进入资源导航界面。生产环境部署请参考 `docs/deployment.md` 文档。

## 贡献指南

1. 在 GitHub Issues 中查找或提交待处理的资源新增请求或功能建议，获得维护者反馈后再行开发，避免重复劳动。

2. 克隆仓库并创建新的功能分支，分支命名遵循 `feature/功能简述` 或 `fix/问题简述` 格式，确保与主分支保持同步。

3. 新增资源链接时，需在 `src/data/schema/` 下补充对应的 JSON 条目，并执行 `npm run validate` 进行格式校验，确保所有必填字段完整。

4. 提交代码前运行 `npm run test` 执行全部测试用例，保证现有功能不被破坏；同时运行 `npm run lint` 统一代码风格。

5. 发起 Pull Request 时需在描述中明确关联 Issue 编号，并附上本地自测结果截图或日志，等待至少一名维护者审核通过后合并。

## 常见问题

Q: 资源链接无法访问时应该如何处理？

A: 项目内置的定时检测任务会自动标记失效链接。用户亦可手动点击资源卡片上的报告失效按钮，系统将在下一次检测周期中优先复检。若确认失效，该资源会被移至归档区并从主列表中隐藏，同时保留历史记录便于后续恢复。

Q: 如何导入自定义的资源列表？

A: 项目支持批量导入 JSON 和 CSV 格式数据。将文件放置于 `data/imports/` 目录下，执行 `npm run import -- --format=csv --path=./data/imports/my-list.csv` 即可。导入前请参考 `docs/data-schema.md` 确认字段映射关系。

Q: 能否将收藏夹数据同步到其他设备？
