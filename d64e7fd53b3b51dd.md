# Yuejiafan Video Resource Index

Yuejiafan Video Resource Index 是一个面向技术内容创作者、教育工作者和开发者的高质量视频资源外链聚合平台。该项目不对视频内容进行二次存储或转码，而是通过系统化的分类索引机制，将分散在 yuejiafan.com 视频频道的优质技术讲解、开源项目演示、架构分析等内容进行统一整理与检索，帮助用户快速定位所需的技术视频资料。

项目定位为技术视频资源导航工具，主要服务于需要持续跟进前沿技术动态、寻找教学素材、或通过视频形式学习复杂技术概念的开发者群体。通过本项目的索引机制，用户可以在数百个视频条目中按主题、难度、技术栈等多维度筛选，显著降低在海量视频内容中查找特定技术资料的时间成本。

## 功能概览

**视频资源索引检索**：提供基于标题关键词、视频ID、技术领域等多字段的组合检索能力，支持模糊匹配与精确查找。

**分类标签体系**：为每个视频条目标注技术领域标签，包括但不限于前端工程化、后端架构设计、DevOps实践、数据科学基础、编程语言特性等，便于用户按技术方向浏览。

**每日更新追踪**：系统定时同步 yuejiafan.com 视频频道的最新发布内容，确保索引库与源站保持实时一致，用户可获取最新的技术视频动态。

**个人收藏与列表**：支持用户将感兴趣的视频条目加入个人收藏列表，并可自定义分类标签进行二次组织，满足个性化学习或教学备课需求。

**快速预览与跳转**：每条索引记录均提供视频标题、发布日期、时长预估、内容简介等元数据，用户确认目标后一键跳转至原始视频播放页。

**开放数据导出**：索引数据支持以 JSON 和 CSV 格式导出，便于教育机构或技术社区将视频资源整合至自有培训平台或学习管理系统。

## 应用场景

技术团队内部培训选材。团队技术负责人可利用本索引按技术栈筛选合适的视频资料，作为内部技术分享会的素材来源或新人入职后的自学参考内容，缩短培训材料的准备周期。

开源项目贡献者学习参考。开源项目维护者可将相关技术领域的视频索引链接整理至项目文档中，帮助贡献者通过视频形式快速理解项目架构设计与核心实现逻辑。

技术社区内容聚合与二次分发。技术社区运营方可通过导出功能将本项目的视频索引数据整合至社区资源库，为社区成员提供结构化的技术视频导航，丰富社区内容生态。

个人开发者系统化学习路径规划。开发者可根据自身技术成长需求，按照分类标签体系筛选对应领域的视频资源，制定系统化的学习计划并利用收藏功能持续追踪学习进度。

## 快速开始

以下命令帮助您在本地环境快速部署并运行 Yuejiafan Video Resource Index 服务。

```
git clone https://github.com/yuejiafan/video-index.git
cd video-index
npm install
npm run build
npm start
```

若使用 Docker 部署，可执行以下命令：

```
docker pull yuejiafan/video-index:latest
docker run -d -p 3000:3000 --name video-index yuejiafan/video-index:latest
```

## 贡献指南

提交 Issue 报告问题或提出功能建议。在提交 Issue 前，请先检索已有 Issue 列表，避免重复报告。描述问题时请提供清晰的重现步骤、预期行为与实际行为对比，并附上运行环境信息（Node.js 版本、数据库版本等）。

Fork 本项目并创建本地功能分支。分支命名规范为 feature/功能简述 或 fix/问题简述，确保分支名称能够清晰表达改动意图。提交代码前请运行测试套件确保全部测试通过。

遵循项目的代码风格与提交规范。代码提交信息格式为 type(scope): subject，type 可选值包括 feat、fix、docs、style、refactor、test、chore，scope 为受影响模块名称。提交前执行 ESLint 与 Prettier 进行代码格式化检查。

提交 Pull Request 至主仓库的 develop 分支。PR 描述中需关联对应的 Issue 编号，并详细说明改动内容、测试覆盖情况及影响范围。PR 合并前需要通过持续集成流程中所有检查项。

## 常见问题

问：项目是否存储或托管视频文件本身？

答：本项目仅存储视频的元数据索引，包括标题、描述、发布时间、视频ID等文本信息，不存储、缓存或代理任何视频文件内容。所有视频播放均跳转至源站 yuejiafan.com 完成。

问：索引数据多久与源站同步一次？

答：系统默认每隔 6 小时执行一次全量同步检查，同时通过源站的内容更新接口实时接收新增视频通知。用户也可在管理后台手动触发立即同步操作。

问：是否支持添加自定义视频源或私有视频链接？

答：当前版本仅支持 yuejiafan.com 官方视频频道的索引。如需添加自定义视频源，可参考开发者文档中的数据源扩展接口，通过实现自定义适配器将第三方视频平台接入索引体系。
