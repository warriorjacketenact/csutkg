# VueBest 技术资源导航

VueBest 是一个面向 Vue.js 生态系统的技术资源导航与精选集项目，旨在为 Vue 开发者提供高质量的外链聚合、学习路径指引和工具推荐。本项目不直接托管代码库或组件，而是以结构化目录的形式，收录并分类整理 Vue 相关的最佳实践、开源项目、教程文章、视频课程以及社区讨论渠道。

本项目主要服务于以下用户群体：正在学习 Vue 框架的初中级开发者、需要快速查找特定场景解决方案的技术负责人、以及希望了解 Vue 生态前沿动态的资深工程师。通过本项目提供的索引结构，用户可以节省大量在互联网中分散检索的时间，直接抵达经过初步筛选的高价值信息节点。

## 功能概览

**精选外链聚合**：围绕 Vue 生态，收集超过 200 个经过人工筛选的优质外链，涵盖官方文档、社区专栏、开源演示项目与工具站点。

**多维度分类索引**：所有资源按照应用场景、技术层级、内容形式三个维度进行交叉分类，支持按需定位。

**每日批次更新机制**：采用批次号管理资源入库进度，当前批次为第 1054/1241 批，确保收录内容的时效性与可追溯性。

**轻量级本地部署**：项目本身为纯静态 Markdown 文档结构，可配合任意静态站点生成器构建为个人导航站。

**社区贡献通道**：开放资源推荐与错误反馈入口，允许开发者提交新的优质链接或报告失效地址。

**无依赖零配置**：无需安装任何后端服务或数据库，克隆仓库后即可在本地浏览或编辑。

## 应用场景

作为 Vue 项目初始化阶段的技术选型参考。当技术负责人需要为新项目选择状态管理方案、UI 组件库或构建工具时，可通过本导航快速定位到相关领域的知名项目首页与对比文章。

作为开发者日常学习的书签替代方案。开发者可以将本项目设为浏览器起始页或收藏夹入口，每日浏览新增资源批次，持续跟踪 Vue 生态的最新动态。

作为技术分享与培训的教学辅助材料。在内部技术分享会或新员工培训中，讲师可直接引用本项目中的资源链接作为课后延伸阅读列表，降低学员自行检索的成本。

## 快速开始

以下指令适用于 macOS、Linux 及 Windows WSL 环境。

```bash
# 克隆项目仓库至本地
git clone https://github.com/vuebest/vuebest-navigator.git

# 进入项目根目录
cd vuebest-navigator

# 安装文档构建依赖（若使用静态站点生成器如 VuePress 或 VitePress）
npm install

# 启动本地开发服务器，预览导航页面
npm run docs:dev
```

若仅需查看资源列表，无需执行安装步骤，直接打开项目根目录下的 `README.md` 或 `docs/index.md` 文件即可。

## 常见问题

如何判断某个外链是否适合收录到本项目中？

收录标准以资源与 Vue 框架的关联强度为首要依据。优先收录以下类型：官方库与工具、社区广泛认可的开源项目、技术会议演讲录像、知名博主撰写的深度解析文章。商业广告、个人博客中无实质技术内容、或仅提及 Vue 但主体内容无关的链接不予收录。具体判定可参考 `docs/contributing/manual.md` 中的收录细则。

本地预览时发现链接无法访问，该如何处理？

首先确认本地网络环境是否可以正常访问该域名。若确认网络无问题，请在 GitHub Issues 中使用 Broken Link 模板提交报告。在修复前，项目维护者会在对应条目中添加 `[待验证]` 标记，并在后续批次更新时重新测试。若连续三次测试均失败，该链接将被移至归档列表。

为什么 README 中直接列出的裸域名或带协议地址不能点击跳转？

本项目遵循开源文档的纯文本兼容性原则，所有外链以原始字符串形式呈现，避免引入 HTML 标签或 Markdown 链接语法对阅读器造成干扰。用户可通过复制粘贴地址至浏览器地址栏访问，或在使用 VitePress 构建的站点中通过插件实现自动链接化。这种设计同时确保了 Markdown 源文件在各种代码托管平台上的渲染一致性。
